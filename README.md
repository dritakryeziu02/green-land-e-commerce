# 🛒 Green Land E-Commerce

A responsive e-commerce website built with **HTML, CSS, and JavaScript**. The application uses the **DummyJSON API** to display products and categories, allowing users to browse products, manage a shopping cart, and complete a simple checkout process.

## ✨ Features

- 🛍️ Browse products by category
- 🔍 View product details
- 🛒 Add and remove items from the shopping cart
- 💾 Cart data stored using Local Storage
- 📱 Responsive design
- 🔗 Dynamic product data fetched from the DummyJSON API

## 🛠️ Technologies

- HTML5
- CSS3
- JavaScript
- DummyJSON API
- Local Storage
